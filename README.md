# Автотесты для страницы EventsWidget

Проект автоматизированного тестирования страницы-конструктора виджета событий `https://dev.3snet.info/eventswidget/`.

## Стек

- Python 3.9
- Playwright
- pytest + pytest-playwright

## Структура проекта

```
├── core/
│   ├── base_page.py      — базовый класс с общими методами (goto, click, fill, expect и др.)
│   ├── constants.py      — константы (дефолтные значения полей, тексты)
│   └── routes.py         — маршруты страниц
├── pages/
│   └── eventswidget_page.py  — Page Object страницы /eventswidget/
├── tests/
│   └── test_eventswidget.py  — тесты
└── conftest.py           — фикстуры (browser context, base_url)
```

## Запуск тестов

**1. Установить зависимости:**

```bash
pip install -r requirements.txt
```

**2. Установить браузеры Playwright:**

```bash
playwright install
```

**3. Запустить тесты:**

```bash
pytest tests/test_eventswidget.py -v --headed
```

Без графического интерфейса (headless):

```bash
pytest tests/test_eventswidget.py -v
```

Запуск конкретного теста:

```bash
pytest tests/test_eventswidget.py::test_default_parameters -v --headed
```

## Покрытие тестами

### `test_eventswidget_loads`
Проверяет что страница открывается и главный баннер отображается.

### `test_default_parameters`
Проверяет дефолтное состояние всех элементов формы при первом открытии страницы:
- дропдаун тематики показывает «Выбрать тематику»
- дропдаун стран показывает «Все страны»
- поле ширины содержит значение `230`
- поле высоты содержит значение `240`
- чекбоксы «на всю ширину» и «на всю высоту» не активированы
- выбранная тема по умолчанию — `turquoise`
- в блоке кода отображается дефолтный iframe-код

### `test_topic_checkboxes_counter`
Проверяет корректность счётчика выбранных тематик:
- при первом открытии счётчик показывает «Выбрать тематику»
- при последовательном выборе каждой темы счётчик увеличивается и соответствует количеству отмеченных чекбоксов
- после нажатия «Очистить» счётчик сбрасывается обратно к «Выбрать тематику»

### `test_minimum_width`
Проверяет ограничение минимальной ширины виджета:
- при вводе 10 случайных значений меньше `230` и нажатии Enter поле автоматически возвращается к минимальному значению `230`

### `test_maximum_width`
Проверяет ограничение максимальной ширины виджета:
- при вводе 10 случайных значений больше `1020` и нажатии Enter поле автоматически возвращается к максимальному значению `1020`

### `test_minimum_height`
Проверяет ограничение минимальной высоты виджета:
- при вводе 10 случайных значений меньше `240` и нажатии Enter поле автоматически возвращается к минимальному значению `240`

### `test_maximum_height`
Проверяет ограничение максимальной высоты виджета:
- при вводе 10 случайных значений больше `720` и нажатии Enter поле автоматически возвращается к максимальному значению `720`

### `test_generation_parameters`
Проверяет корректность генерации iframe-кода виджета:
- выбирается случайная тематика из доступных
- задаётся случайная ширина (230–1020) и высота (240–720)
- выбирается случайная цветовая тема (blue, green, turquoise, purple)
- после нажатия «Сгенерировать превью» проверяется что:
  - в сгенерированном коде (`textarea`) присутствуют все выбранные параметры: `theme`, `width`, `height`
  - атрибуты `<iframe>` в превью (`width`, `height`, `src`) соответствуют выбранным значениям

## Замеченные баги и спорное поведение

**Баги:**

- **Счётчик выбранных стран** — дропдаун стран присутствует, но выбор страны недоступен. При этом если нажать «Выбрать все» счётчик меняется на «Выбрано: 1» вместо ожидаемого «Все страны».
- **Тематика «Финтех» написана по-русски** — при переключении сайта на английский язык название меняется на «Fintech». Это несоответствие указывает на локализационную ошибку в русской версии.

**Спорное поведение:**

- **Чекбоксы «на всю ширину контейнера» и «на всю высоту блока»** — при их активации инпут блокируется (`disabled`), однако значение в поле не обновляется. Возможно это задумано, но вводит пользователя в заблуждение.

## Что не покрыто тестами

Текущие тесты охватывают далеко не все возможные сценарии. Среди непокрытых:

- **Смена языка** — не проверяется корректность локализации всех элементов при переключении на английский язык и обратно
- **Выбор страны** — дропдаун стран не протестирован, счётчик выбранных стран не проверяется
- **Генерация кода при разных комбинациях параметров** — не покрыты сценарии с одновременным выбором нескольких тематик и стран
- **Чекбоксы «на всю ширину» и «на всю высоту»** — не проверяется что после их активации в генерируемом коде меняются соответствующие атрибуты
- **Смена цветовой темы и проверка превью** — не проверяется визуальное отражение выбранной темы в превью
- **Назначение баннера и гифки** — непонятно какую функцию они выполняют и должны ли они меняться в зависимости от настроек виджета, тесты на это отсутствуют
- **Кнопка «Скопировать код»** — не проверяется что код действительно копируется в буфер обмена
- **Валидация граничных значений** — не покрыты значения ровно на границе (230, 240, 1020, 720)
